# Sample Projects

## Local RAG Evidence Engine

Built a local retrieval augmented generation system over Markdown and PDF
sources. The system chunked documents, generated embeddings, returned cited
answers, and included a fallback answer path when no LLM key was configured.

## Agent Memory Prototype

Implemented persistent memory for an AI coding assistant using JSON storage,
BM25 retrieval, and token-budgeted memory injection.

## Evaluation Harness

Created deterministic JSON result contracts and tests for LLM-powered tools,
including retrieval metrics and contract validation.
