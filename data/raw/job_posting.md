# AI Engineer - Retrieval and Agent Evaluation

We are looking for an AI Engineer to build reliable LLM applications in Python.
The role involves retrieval augmented generation, agent memory, evaluation
pipelines, SQL or analytics workflows, and production-quality testing.

Responsibilities include building document ingestion pipelines, improving
retrieval quality, designing JSON output contracts, writing pytest coverage,
and explaining tradeoffs with metrics such as Recall@k, MRR, and nDCG.

Nice to have: experience with ChromaDB or vector stores, BM25, embeddings,
LiteLLM, structured outputs, and local-first AI tooling.
